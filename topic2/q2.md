## 요구사항

1. "피치클락 위반 시 타자와 투수에게 각각 어떤 페널티가 부여되나요?"와 같은 질문에 대해 정확한 규정 번호와 내용을 인용하여 답변하는 챗봇을 만드시오.
2. "A선수가 3일 전 경기에 나갔고 오늘 부상으로 말소됐다면, 소급 적용을 포함해 언제 복귀 가능한가?"라는 질문에 대해 ID 8, 9, 10의 규정을 논리적으로 조합해 날짜를 계산하는 체인을 구현하시오.
3. "경기 중 ABS가 고장 났고, 5회초에 비가 내려 경기가 중단된 상황"을 가정합니다. 이때 **ID 2(ABS 실패), ID 24(강우콜드), ID 23(서스펜디드)**를 모두 참조하여 현재 경기의 최종 상태(종료/일시중단)를 판정하는 논리 체인을 구성하시오.
